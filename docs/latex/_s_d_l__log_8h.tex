\subsection{Detailed Description}
Simple log messages with categories and priorities.

By default logs are quiet, but if you're debugging S\-D\-L you might want\-: \begin{DoxyVerb}SDL_LogSetAllPriority(SDL_LOG_PRIORITY_WARN);
\end{DoxyVerb}


Here's where the messages go on different platforms\-: Windows\-: debug output stream Android\-: log output Others\-: standard error output (stderr)
