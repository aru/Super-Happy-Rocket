32  * This class allows you to simply add triangles as if this class were a
33  * container. The AddTriangle() function searches the current list of triangles
34  * and determines if the vertex/normal/texcoord is a duplicate. If so, it addes
35  * an entry to the index array instead of the list of vertices.
36  * When finished, call EndMesh() to free up extra unneeded memory that is reserved
37  * as workspace when you call BeginMesh().
38  *
39  * This class can easily be extended to contain other vertex attributes, and to
40  * save itself and load itself from disk (thus forming the beginnings of a custom
41  * model file format).
42  *
43  */
